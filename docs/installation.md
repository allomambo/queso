# Installation

## Install the package

::: code-group

```sh [yarn]
$ yarn add @allomambo/queso
```

```sh [npm]
$ npm i @allomambo/queso
```

:::

## Sass core

Import the sass core from the package in your project root file, often `App.vue`

```vue
<script setup>
import "@allomambo/queso/styles/core";
</script>
```
